# UX Wireframes – V1

## ניווט ראשי במובייל
תפריט תחתון עם 5 טאבים:
1. Home
2. Groups
3. Chats
4. Notifications
5. Profile

## עקרונות UX
- מהיר, דיסקרטי ולא חושפני
- מעט אפשרויות, הרבה בהירות
- קהילה במרכז
- פעולות רגישות דורשות אישור
- פרטיות נגישה בקליק

## מסכי ליבה
### 1. דף נחיתה ציבורי
- לוגו
- כפתור התחברות / הרשמה
- Hero עם מסר על קהילה דיסקרטית
- יתרונות: קהילה, קבוצות סגורות, חיבורים בהסכמה, משתמשים מאומתים
- הסבר קצר על פרטיות, אימות ובקרה
- CTA להרשמה

### 2. הרשמה
- אימייל / טלפון
- סיסמה
- אישור סיסמה
- אישור תנאי שימוש
- אישור 18+
- מעבר לאימות

### 3. בניית פרופיל
#### שלב 1
- שם תצוגה
- תיאור קצר
- תחומי עניין
- אזור כללי

#### שלב 2 – תמונות
- תמונת פרופיל
- גלריית תמונות
- הגדרת חשיפה לתמונת פרופיל
- הגדרת חשיפה לגלריה

#### שלב 3 – פרטיות
- מי רואה את הפרופיל
- מי יכול לשלוח בקשת צ'אט
- האם לאפשר בקשות גישה לתמונות
- האם להציג סטטוס פעילות
- האם להציג תג מאומת

### 4. בית
- Greeting קצר
- שדה יצירת פוסט
- פילטרים לפיד
- כרטיסי פוסטים
- קבוצות מומלצות
- אירוע מקודם

### 5. יצירת פוסט
- שדה טקסט
- העלאת תמונה
- בחירת יעד: פיד כללי / קבוצה
- בחירת קטגוריה / תגית
- חיווי האם הפרסום גלוי או אנונימי

### 6. רשימת קבוצות
- חיפוש
- פילטרים: שלי / מומלצות / פתוחות / סגורות
- כרטיסי קבוצות
- סטטוס הצטרפות

### 7. קבוצה בודדת
- שם + תיאור + כללים
- סוג קבוצה
- כפתור הצטרפות / בקשה להצטרפות
- פיד פנימי
- מנהלי קבוצה

### 8. פרופיל שלי
- תמונת פרופיל
- שם תצוגה
- תג מאומת
- bio
- תחומי עניין
- אזור
- גלריה
- עריכת פרופיל
- עריכת פרטיות
- הגדרות חשבון ואבטחה

### 9. פרופיל של משתמש אחר
- תמונת פרופיל
- שם תצוגה
- תג מאומת
- bio
- תחומי עניין
- אזור
- תמונות גלויות לפי הרשאה
- כפתור בקשת צ'אט
- כפתור בקשת גישה לגלריה
- דיווח / חסימה

### 10. בקשות צ'אט
- לשונית נכנסות
- לשונית יוצאות
- אישור / דחייה

### 11. שיחות
- רשימת שיחות
- חיפוש

### 12. צ'אט פרטי
- Header עם שם תצוגה ותג מאומת
- בועות שיחה
- שדה כתיבה
- פעולות: דיווח / חסימה / סיום שיחה

### 13. התראות
- פילטרים: הכול / קהילה / קבוצות / צ'אטים / מערכת
- סימון הכול כנקרא

### 14. אירוע מקודם
- תמונה ראשית
- כותרת
- תיאור
- קוד קופון
- כפתור מעבר

### 15. הגדרות פרטיות
- מי רואה את הפרופיל
- מי רואה את תמונת הפרופיל
- מי רואה את הגלריה
- מי יכול לשלוח בקשת צ'אט
- האם ניתן לבקש גישה לתמונות
- האם להציג סטטוס פעילות
- האם להציג תג מאומת

### 16. הגדרות חשבון ואבטחה
- שינוי סיסמה
- שינוי אימייל / טלפון
- ניהול סשנים
- יציאה מכל המכשירים
- מחיקת חשבון / השבתה זמנית
- Theme preference: light / dark
